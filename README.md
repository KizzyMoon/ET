# Event Team Hub

A small GitHub Pages website for managing event team members, meetings, attendance, and meeting notes.

## Use It

Open `index.html` in a browser, or host this folder with GitHub Pages.

The site saves data in the browser on the device being used. Use the download button in the header to back up the data, and the upload button to restore it later.

## GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, and `README.md`.
3. In the repository, go to Settings, then Pages.
4. Set the source to the `main` branch and the root folder.
5. Save, then open the published Pages link when GitHub finishes building.
